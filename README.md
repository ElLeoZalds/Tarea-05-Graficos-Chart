# Proyecto Gráficos y Búsqueda DNI/RUC

## Descripción
Aplicación web que genera gráficos con ChartJS y permite búsquedas de DNI y RUC usando una API.

## Funcionalidades

### Gráficos
- Gráfico 1: Lenguajes de programación con valores ingresados.
- Gráfico 2: Días de la semana con datos desde JSON.

### Búsqueda DNI
- Busca persona por DNI.
- Muestra nombres y apellidos.
- API de :contentReference[oaicite:0]{index=0}.

### Búsqueda RUC
- Busca empresa por RUC.
- Muestra razón social, estado y dirección.
- API de :contentReference[oaicite:1]{index=1}.

- JavaScript
